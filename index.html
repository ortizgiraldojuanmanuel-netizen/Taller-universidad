Corrección de errores (Parte 3)

Errores originales y corrección:

  • <h1>Mi nombre<h1> → <h1>Mi nombre</h1>
  • <p>Este es un párrafo<p> → <p>Este es un párrafo</p>
  • <img src=imagen.jpg> → <img src="imagen.jpg" alt="Imagen">
  • colr: blue; → color: blue;
  • font sixe: 16px; → font-size: 16px;

Quiz Creativo

  1. <ul> crea listas desordenadas y <ol> listas ordenadas.
  2. border-radius redondea las esquinas.
  3. background-color cambia el color de fondo.
  4. Comentario en HTML: <!-- texto -->
  5. <header> define la cabecera.
  6. text-align: center; centra el texto.
  7. id es único, class puede repetirse.
  8. <footer> representa el pie de página.
  9. <img> inserta una imagen.
  10. <p> es un párrafo.
  11. Lista ordenada con <ol>.
  12. Hoja de estilos: <link rel="stylesheet" href="estilos.css">
  13. color cambia el texto.
  14. text-align: center; centra texto.
